# Bac Bo Tie Bot

Frontend em Next.js para registrar resultados de Bac Bo, aplicar estratégia manual de Tie, controlar banca, stop-loss e stop-gain.

> Este projeto não prevê resultados e não automatiza apostas em plataformas reais. É uma ferramenta manual de registro, análise e gestão de risco.

## Rodar localmente

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Subir para GitHub

```bash
git init
git add .
git commit -m "Initial Bac Bo Tie Bot frontend"
git branch -M main
git remote add origin https://github.com/SEU_USUARIO/bacbo-tie-bot.git
git push -u origin main
```

## Deploy na Vercel

1. Acesse Vercel.
2. Clique em New Project.
3. Importe o repositório do GitHub.
4. Deploy.
